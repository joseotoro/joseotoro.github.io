The Fibonacci sequence:
$F_n = F_{n-1} + F_{n-2}$
with:
$F_0 = 0, F_1 = 1$

Code (in Scala): {% highlight scala %} def fibonacci(n: Int): Int = { n match { case 0 => 0 case 1 => 1 case _ => fibonacci(n - 1) + fibonacci(n - 2) } } {% endhighlight %}
